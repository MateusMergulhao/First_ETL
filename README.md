# First ETL

Este repositório contém meu primeiro projeto de ETL (Extract, Transform, Load) desenvolvido em Python.

O objetivo deste projeto é aprender na prática como funciona um pipeline ETL, realizando a extração de dados de um arquivo CSV, aplicando etapas de transformação e limpeza dos dados, e por fim carregando essas informações em um banco de dados MongoDB Atlas.

Este projeto faz parte do meu processo de aprendizado em engenharia de dados e manipulação de dados, explorando conceitos importantes como leitura de arquivos, tratamento de dados e integração com bancos de dados NoSQL.

# Tecnologias Utilizadas

- Python

- Pandas

- MongoDB Atlas

- PyMongo

# Como Executar o Projeto

1. Clonar o repositório:
git clone https://github.com/MateusMergulhao/First_ETL.git

2. Instalar dependências:
pip install pandas pymongo

3. Configurar a conexão com o MongoDB:
No arquivo config.py, adicione sua string de conexão do MongoDB Atlas.
